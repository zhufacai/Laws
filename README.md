# Laws

## 分支说明

```text
Laws
  ├── main       # md 格式数据
  └── docx       # docx 等非结构化数据
```

> [!TIP]
>
> - main 分支作为 EnforcementBook-Web，EnforcementBook 项目数据源
> - docx 分支作为源数据的归档存储，暂未实际作用

Markdown 格式文件更新，可以遵照 Docusaurus 的 [Markdown 特性](https://docusaurus.io/zh-CN/docs/markdown-features) 来整理，使其 EnforcementBook-Web 看起来更加简介和美观；如果还需要让 EnforcementBook 项目完美支持，那就只能使用最基本的 markdown 格式（[Markdown 语法速查表](https://www.markdownlang.com/zh/cheatsheet/)）定义文件内容

> [!Waring]
>
> - md 文件中 < 使用 `&lt;` > 使用 `&gt;` 字符替换，比如：`<通知>` 修改后为 `&lt;通知&gt;`
> - docx 分支作为源数据的归档存储，暂未实际作用
